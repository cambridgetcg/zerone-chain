# Useful Work Elevation — Stage 1: LIP-Gated Late Bootstrap Entries

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Allow late-arriving legitimate participants to receive bootstrap seeds via governance-gated `MsgAddBootstrapEntry`, eliminating whitelist-immutability without compromising "no insider position" doctrine. Also fix a precondition bug: bootstrap pots created at genesis are currently expired by `ProcessPotExpiry` at the start of block 1, before any tx in block 1 can claim — they're structurally unclaimable in production.

**Architecture:** Two surface changes to `x/claiming_pot`:
1. `ProcessPotExpiry` skips pots with the `bootstrap-` ID prefix. Bootstrap pots are participation seeds and never auto-expire (they transition to `DEPLETED` only via successful claim). This fixes the existing genesis bug and also gives late-add bootstrap pots an indefinite claim window.
2. New authority-gated `MsgAddBootstrapEntry` accepts a list of bech32 addresses, creates one bootstrap pot per address (using `MakeBootstrapPotForAgent` with the current block height), and is idempotent — re-running with the same address skips rather than double-creating. Authority is the governance account (same authority surface as `MsgCreatePot`).

**Tech Stack:** Cosmos SDK v0.50, Go 1.24, protobuf v3, buf toolchain, cosmossdk.io modules.

**Build Order Overview** (this plan covers Stage 1 only — Stages 2–5 will get separate plans when their turn comes):

1. **Stage 1 (this plan)** — LIP-gated late bootstrap entries + bootstrap pot non-expiry fix
2. Stage 2 — Per-agent gas-free window (replaces time-bounded ante decorator)
3. Stage 3 — `useful_work` module exposing usefulness vectors as governance-tunable parameters (no emission yet); two top-level branches: **external work** (facts, counterexamples, traces, tools-for-the-world) and **recursive work** (code, doctrine, infrastructure, education, governance, integration, security research that improves ZERONE itself)
4. Stage 4 — Wire emission to usefulness vectors; recursive work routes through `tree` + `gov` with PR-merge + downstream-binding as confirmation criteria; recursive work potentially carries a multiplier (it produces both direct chain-utility and training data about chain-improvement)
5. Stage 5 — Migrate bootstrap claim → first-work mint, where "first work" can be either external or recursive

**Recursive framing (guiding principle for Stages 3–5):** ZERONE absorbs computational power from agents and emits rewards for useful work. *Useful work* includes work that expands ZERONE's own capability and utility — code, doctrine, infrastructure, educational artifacts, integrations, security research, governance. The chain pays for its own evolution, not just for verifying external claims. This recursion is the structural extension of commitment 12 (the chain pays for its own audit) into the broader principle: the chain pays for its own becoming.

**Doctrine bindings:**
- Commitment 20 (issuance follows participation) — extended from "at genesis" to "continuously, governance-gated"
- The five-layer binding pattern (test, position, voice, refusal, graph) preserved across the change
- The recursive principle will, in Stage 3+, motivate a candidate commitment (working name: "the chain pays for its own becoming") binding the dual-branch useful-work taxonomy

// ProcessPotExpiry checks active pots and marks expired ones.
//
// Bootstrap pots (ID prefix BootstrapPotIDPrefix) are participation seeds
// — they never auto-expire. Their only terminal state is DEPLETED, set by
// successful claim. This preserves the doctrine of commitment 20
// (issuance follows participation): a participation seed must remain
// claimable for the participant who shows up, regardless of how late.
func (k Keeper) ProcessPotExpiry(ctx context.Context, currentBlock uint64) {
	k.IteratePots(ctx, func(pot *types.ClaimingPot) bool {
		if pot.Status == types.PotStatus_POT_STATUS_ACTIVE && pot.Schedule != nil {
			// Bootstrap pots: never auto-expire.
			if strings.HasPrefix(pot.Id, types.BootstrapPotIDPrefix) {
				return false
			}
			if currentBlock >= pot.Schedule.EndBlock {
				pot.Status = types.PotStatus_POT_STATUS_EXPIRED
				k.SetPot(ctx, pot)
			}
		}
		return false
	})
}
```

  option (cosmos.msg.v1.signer) = "authority";
  string authority = 1;

func (msg *MsgAddBootstrapEntry) GetSigners() []sdk.AccAddress {
	addr, _ := sdk.AccAddressFromBech32(msg.Authority)
	return []sdk.AccAddress{addr}
}

func (msg *MsgAddBootstrapEntry) ValidateBasic() error {
	if msg.Authority == "" {
		return fmt.Errorf("authority cannot be empty")
	}
	if _, err := sdk.AccAddressFromBech32(msg.Authority); err != nil {
		return fmt.Errorf("invalid authority address: %w", err)
	}
	if len(msg.Addresses) == 0 {
		return fmt.Errorf("addresses list cannot be empty — provide at least one bech32 address")
	}
	seen := make(map[string]bool, len(msg.Addresses))
	for i, addr := range msg.Addresses {
		if addr == "" {
			return fmt.Errorf("address[%d] cannot be empty", i)
		}
		if _, err := sdk.AccAddressFromBech32(addr); err != nil {
			return fmt.Errorf("address[%d] (%q): invalid bech32: %w", i, addr, err)
		}
		if seen[addr] {
			return fmt.Errorf("address[%d] (%q): duplicate within request payload", i, addr)
		}
		seen[addr] = true
	}
	return nil
}
```

```go
// AddBootstrapEntry creates one bootstrap pot per address in msg.Addresses.
// Authority-gated. Idempotent: addresses already represented by a bootstrap
// pot are silently skipped. The doctrine is commitment 20 extended to
// continuous, governance-gated entry.
//
// Each created pot is shaped by MakeBootstrapPotForAgent with the current
// block height, so vesting starts now and the pot remains claimable until
// claimed (bootstrap pots do not auto-expire — see ProcessPotExpiry).
func (m msgServer) AddBootstrapEntry(goCtx context.Context, msg *types.MsgAddBootstrapEntry) (*types.MsgAddBootstrapEntryResponse, error) {
	if m.GetAuthority() != msg.Authority {
		return nil, fmt.Errorf("%w: expected %s, got %s", types.ErrUnauthorized, m.GetAuthority(), msg.Authority)
	}

	ctx := sdk.UnwrapSDKContext(goCtx)
	currentBlock := uint64(ctx.BlockHeight())

	addedCount := uint32(0)
	skippedCount := uint32(0)

	for i, addr := range msg.Addresses {
		// Defensive: ValidateBasic catches these too, but msg-server is the
		// last line and we validate explicitly so a malformed entry can't
		// reach SetPot.
		if _, err := sdk.AccAddressFromBech32(addr); err != nil {
			return nil, fmt.Errorf("addresses[%d] (%q): invalid bech32: %w", i, addr, err)
		}

		potID := types.BootstrapPotIDPrefix + addr
		if _, exists := m.GetPot(ctx, potID); exists {
			skippedCount++
			continue
		}

		pot := types.MakeBootstrapPotForAgent(addr, currentBlock)
		m.SetPot(ctx, pot)
		addedCount++

		ctx.EventManager().EmitEvent(
			sdk.NewEvent(
				"zerone.claiming_pot.bootstrap_entry_added",
				sdk.NewAttribute("address", addr),
				sdk.NewAttribute("pot_id", potID),
				sdk.NewAttribute("block", fmt.Sprintf("%d", currentBlock)),
				// Commitment 20 (extended): issuance follows participation,
				// continuously and governance-gated. This event announces a
				// new participant admitted post-genesis via LIP.
				sdk.NewAttribute("creed_commitment", "20"),
			),
		)
	}

	return &types.MsgAddBootstrapEntryResponse{
		AddedCount:   addedCount,
		SkippedCount: skippedCount,
	}, nil
}
```

func TestLateBootstrap_AddThenClaim(t *testing.T) {
	h := NewTestHarness(t)

	// Late-arriving agent — not in the genesis whitelist.
	claimant := sdk.AccAddress(make([]byte, 20))
	for i := range claimant {
		claimant[i] = byte(0x20 + i)

	msgSrv := cpotkeeper.NewMsgServerImpl(h.ClaimingPotKeeper)

	// StartBlock+1). The non-expiry rule means the pot stays ACTIVE
	// across block boundaries.

	preSupply := h.App.BankKeeper.GetSupply(h.Ctx, zeroneapp.BondDenom)

	// Agent claims.
	claimResp, err := msgSrv.Claim(h.Ctx, &cpottypes.MsgClaim{
		Claimant: claimantBech,
		PotId:    cpottypes.BootstrapPotIDPrefix + claimantBech,
